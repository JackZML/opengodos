## 🏗️ 系统架构

### 核心组件
1. **神经元系统** (`src/core/neuron.py`)
   - 神经元基类，支持多种神经元类型
   - 状态管理、信号处理、生命周期

2. **信号系统** (`src/core/signal.py`)
   - 信号类型：EXCITATORY, INHIBITORY, MODULATORY
   - 信号优先级和路由
   - 复合信号和条件信号

3. **运行时引擎** (`src/core/runtime_engine.py`)
   - 神经元调度器
   - 思维步循环
   - 状态监控和数据收集

4. **AI集成系统** (`src/ai/llm_service.py`)
   - 多AI提供商支持
   - 自动降级模式
   - 统一的AI调用接口

5. **Web界面** (`web/app.py`)
   - 实时状态监控
   - REST API + WebSocket
   - 类似OpenClaw的用户体验

### 特色功能
1. **生物启发架构**: 神经元拓扑连接，智能自然涌现
2. **声明式开发**: YAML描述文件，AI生成代码
3. **混合智能系统**: 规则 + AI，确保系统始终可用
4. **实时监控**: Web控制台，完全可观测
