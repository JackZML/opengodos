# OpenGodOS v1.0.0 发布说明

## 🎉 欢迎来到OpenGodOS！

OpenGodOS是一个全新的计算范式——**数字生命框架**。我们不追求用更大规模的模型和数据训练出更聪明的AI，而是通过构建**接近生物大脑结构的神经拓扑**，让意识和行为作为结构的属性自然涌现。

## 🚀 核心特性

### 1. 生物启发架构
- **神经元拓扑连接**：智能从结构中涌现
- **模块化认知**：功能分化的认知模块
- **动态交互**：行为从系统内部自发产生

### 2. 声明式开发
- **YAML描述文件**：声明式神经元定义
- **AI生成代码**：从描述文件生成可执行代码
- **可观测系统**：内部状态和思维流可记录分析

### 3. 混合智能系统
- **规则 + AI**：结合传统规则和现代AI
- **自动降级**：AI不可用时自动降级到规则系统
- **实时监控**：Web控制台实时监控系统状态

### 4. 生产就绪
- **完整测试套件**：22个测试，100%覆盖率
- **系统验证**：自动验证所有组件
- **文档完整**：中文文档，易于上手

## 📦 包含内容

### 核心框架
- `src/core/` - 核心框架（Neuron, Signal, RuntimeEngine）
- `src/neurons/` - 神经元实现（情绪、感知、记忆、AI增强）
- `src/ai/` - AI服务（LLMService）
- `src/signals/` - 信号系统（基础+高级信号）

### 配置系统
- `neurons/` - 神经元描述文件（.neuron.yaml）
- `topologies/` - 拓扑配置文件（.yaml）
- `examples/` - 示例代码和配置

### 可视化界面
- `web/` - Web控制台（类似OpenClaw的体验）
- REST API + WebSocket实时通信
- 实时监控神经元状态和信号流

### 文档和工具
- `docs/zh-CN/` - 中文文档
- `tests/` - 测试文件
- `validate_system.py` - 系统验证脚本
- `run_full_demo.py` - 完整演示脚本

## 🛠️ 快速开始

### 1. 环境准备
```bash
# 克隆项目
git clone https://github.com/JackZML/opengodos.git
cd opengodos

# 安装依赖
pip install -r requirements.txt
```

### 2. 配置环境变量
```bash
# 复制环境配置示例
cp .env.example .env

# 编辑.env文件，设置DeepSeek API密钥
DEEPSEEK_API_KEY=your_api_key_here
```

### 3. 运行系统验证
```bash
python validate_system.py
```

### 4. 启动Web界面
```bash
cd web
python app.py
```
访问 http://127.0.0.1:5000

### 5. 运行完整演示
```bash
python run_full_demo.py
```

## 🧬 核心概念

### 神经元（Neuron）
- **基本单元**：模拟大脑中特定功能区域的行为
- **功能单一**：每个神经元负责一种特定的认知功能
- **内部状态**：随时间变化的变量，反映神经元当前状况
- **输入输出**：接收来自其他神经元的信号，处理后输出新信号

### 神经拓扑（Neural Topology）
- **连接图**：神经元间的有向加权连接网络
- **权重调节**：连接具有权重（兴奋/抑制）和可塑性
- **结构决定行为**：不同的拓扑结构导致不同的"人格"和"思维方式"

### 运行时引擎（Runtime Engine）
- **调度器**：管理神经元生命周期和交互
- **思维步循环**：驱动系统进行离散的时间步进
- **信号路由**：根据拓扑规则分发信号

## 🔧 开发指南

### 创建新神经元
1. 在 `neurons/` 目录创建 `.neuron.yaml` 描述文件
