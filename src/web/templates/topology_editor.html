基本操作

  • 拖放元素: 从左侧面板拖放元素到画布
  • 选择元素: 点击元素进行选择，按住Shift多选
  • 移动元素: 拖动选中的元素
  • 连接神经元: 使用连接工具，点击源神经元再点击目标神经元
  • 删除元素: 选择元素后按Delete键或使用删除工具

快捷键

Ctrl + Z 撤销
Ctrl + Y 重做
Ctrl + S 保存
Ctrl + O 打开
Delete 删除选中
Space 平移工具
C 连接工具
V 选择工具

元素类型

  • 标准神经元: 基本处理单元
  • 输入神经元: 接收外部输入
  • 输出神经元: 产生输出结果
  • 隐藏神经元: 中间处理单元
  • 记忆神经元: 具有记忆功能
  • 兴奋性连接: 增强目标神经元活动
  • 抑制性连接: 抑制目标神经元活动
  • 调节性连接: 调节连接强度
