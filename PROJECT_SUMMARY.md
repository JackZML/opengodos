## 🧬 技术架构

### 核心组件
1. **神经元系统** (Neuron)
   - 基类系统，支持多种神经元类型
   - 状态管理、信号处理、生命周期
   - 工厂模式创建不同类型神经元

2. **信号系统** (Signal)
   - 神经元间通信机制
   - 多种信号类型：兴奋、抑制、调节等
   - 优先级和生存时间管理

3. **运行时引擎** (RuntimeEngine)
   - 调度神经元执行
   - 管理信号路由
   - 系统状态监控

4. **AI集成系统** (LLMService)
   - 统一AI调用接口
   - 支持多种AI提供商
   - 降级模式确保可靠性

5. **Web可视化界面**
   - Flask Web应用
   - 实时状态监控
   - REST API + WebSocket

### 关键技术特性
1. **声明式开发**: YAML描述文件生成代码
2. **混合智能**: 规则 + AI，自动降级
3. **实时监控**: Web控制台实时查看
4. **生产就绪**: 完整测试和验证
5. **可扩展架构**: 易于添加新功能
